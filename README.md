# SafeParking Application C# MySQL

Simple application to manage your actual parking clients.

## Specifications

Framework: .Net 4.8

## How to use

1. Simply clone this repository and upload MySQL database:

        git clone https://github.com/skorupcia/SafeParking

2. Enjoy!
